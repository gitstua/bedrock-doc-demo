# Bedrock Knowledge Base Frontend

A modern Vue 3 + Vite application for interacting with the Bedrock Knowledge Base.

## Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`.

## Features

- **Modern UI**: Dark mode with premium glassmorphism aesthetics.
- **Real-time Chat**: Interaction with Bedrock KB via the FastAPI backend.
- **Responsive Design**: Works on Desktop and Mobile.
